# 项目创建
---
1. 创建项目
```bash
cnpm install create-react-app -g
```
```bash
create-react-app react-antd-cms
```
```bash
cd react-antd-cms
```
```bash
npm start
```
2. 打包上线
```bash
npm run build 
```
3. webpack项目文件配置
- 在项目的根目录下执行以下命令：
```bash
git init
git add --all
git commit -m 'tip message'
npm run eject
```
